二维费用背包

分组背包

有 nnn 件物品和一个大小为 mmm 的背包，第 iii 个物品的价值为 w[i]w[i]w[i]，体积为 v[i]v[i]v[i].
同时，每个物品属于一个组，同组内最多只能选择一个物品。求背包能装载物品的最大总价值.
其实是从「在所有物品中选择一件」变成了「从当前组中选择一件」，于是就对每一组进行一次 0−10-10−1 背包就可以了。

这里要注意：一定不能搞错循环顺序，这样才能保证正确性。

有依赖的背包

这些有依赖的背包，可以变成分组背包来做

P1064 [NOIP2006 提高组] 金明的预算方案

金明有 nnn 元钱，想要买 mmm 个物品，第 iii 件物品的价格为 v[i]v[i]v[i]，重要度为 p[i]p[i]p[i]。有些物品是从属于某个主件物品的附件，要买这个物品，必须购买它的主件。
目标是让所有购买的物品的 v[i]∗p[i]v[i] * p[i]v[i]∗p[i] 之和最大。
解法:考虑分类讨论。对于一个主件和它的若干附件，有以下几种可能：只买主件，买主件 + 某些附件。因为这几种可能性只能选一种，所以可以将这看成分组背包。

如果是多叉树的集合，则要先算子节点的集合，最后算父节点的集合。

P3961 [TJOI2013] 黄金矿工

题目描述
小 A 最近迷上了在上课时玩《黄金矿工》这款游戏。为了避免被老师发现，他必须小心翼翼，因此他总是输。
在输掉自己所有的金币后，他向你求助。每个黄金可以看做一个点（没有体积）。现在给出你 N 个黄金的坐标，挖到它们所需要的时间以及它们的价值。有些黄金在同一条直线上，这时候你必须按顺序挖。你可以瞬间把钩子转到任意角度。
小 A 一开始在坐标 (0,0)(0, 0)(0,0)，请你帮助小 A 算出在时间 TTT 内他最多可以得到多少价值的金子。
范围: N(200)N(200)N(200) T(4e4)T(4e4)T(4e4)

P1450 [HAOI2008] 硬币购物

背包+容斥
先考虑没有个数限制的情况，f[i]f[i]f[i] 表示没有限制的方案数;
答案为 f[s]−f[s]-f[s]− 不合法的情况;
考虑一种硬币超过限制时的方案数为 f[s−(次硬币的总数+1)×此硬币的价值]f[s-(\text{次硬币的总数}+1)\times \text{此硬币的价值}]f[s−(次硬币的总数+1)×此硬币的价值];
然后有重复,考虑对不合法的情况进行容斥.
