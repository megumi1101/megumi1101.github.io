AGC003C BBuBBBlesort!

题意：给定一个长度为 NNN 的互异整数序列 AAA。你可以进行两种操作：交换相邻两个元素，或反转连续三个元素。目标是把序列变成升序，求最少需要使用多少次相邻交换操作。

数据范围：1≤N≤1051 \le N \le 10^51≤N≤105，0≤Ai≤1090 \le A_i \le 10^90≤Ai​≤109，且所有 AiA_iAi​ 两两不同。

题解：

Trick or Hint： 观察

注意到反转连续三个元素，它不会改变任何元素所在位置的奇偶性。
也就是说，只靠操作 222，我们可以在奇数位内部和偶数位内部分别随便调整顺序。

所以真正需要操作 111 的地方，只是那些排序后位置奇偶性和当前所在位置奇偶性不同的元素。
如果从 111 开始离散化之后
答案就是奇数目前在偶数位置的个数。

AGC003D Anticube

题意：给定 NNN 个正整数 s1,…,sNs_1,\ldots,s_Ns1​,…,sN​，可以从中选出若干个数。要求任意两个被选中的不同位置 i,ji,ji,j 都满足 sisjs_is_jsi​sj​ 不是立方数。求最多能选出多少个数。

数据范围：1≤N≤1051 \le N \le 10^51≤N≤105，1≤si≤10101 \le s_i \le 10^{10}1≤si​≤1010。

题解：

Trick or Hint： 观察

先把每个数中的立方因子都去掉。这个操作不会影响两个数的乘积是否为立方数。

化简之后，每个质因子的指数只会是 0,1,20,1,20,1,2。于是对一个数 xxx，如果想找到一个数 yyy 使得 xyxyxy 是立方数，那么 yyy 的形式是唯一确定的：指数为 111 的质因子需要补 222 次，指数为 222 的质因子需要补 111 次。

接下来把每个化简后的数和它的互补数配成一组。对于一组互补的数，只能选择其中出现次数更多的一边；特殊地，111 只能选一次。

分解质因数时不能直接对每个数做到根号。可以先用不超过 1010/310^{10/3}1010/3 的质数并试除它们；剩下的部分至多只会包含两个质数。再判断它是平方数、还是一个或两个大质数的乘积，然后找到互补数就行。

复杂度 O(Nπ(1010/3)+Nlog⁡N)O(N\pi(10^{10/3}) + N\log N)O(Nπ(1010/3)+NlogN)。

AGC003E Sequential operations on Sequence

题意：初始序列长度为 NNN，第 iii 个元素为 iii。接下来进行 QQQ 次操作，第 iii 次操作给定 qiq_iqi​，把当前序列无限重复后取前 qiq_iqi​ 项作为新序列。求所有操作结束后，数字 111 到 NNN 各出现多少次。

数据范围：1≤N≤1051 \le N \le 10^51≤N≤105，0≤Q≤1050 \le Q \le 10^50≤Q≤105，1≤qi≤10181 \le q_i \le 10^{18}1≤qi​≤1018。

题解：

Trick or Hint： 倒推

这种前缀反复截取的问题，很自然可以倒推。

如果后一次操作的长度不超过前一次，那么前一次操作不会影响最终序列，可以直接删掉。
于是保留下来的长度一定严格递增，记为 a0=N,a1,…,ama_0=N,a_1,\ldots,a_ma0​=N,a1​,…,am​。

倒推时，设当前有 cnticnt_icnti​ 个长度为 aia_iai​ 的完整块。
它会拆成 ⌊ai/ai−1⌋\lfloor a_i/a_{i-1}\rfloor⌊ai​/ai−1​⌋ 个完整的 ai−1a_{i-1}ai−1​，以及一个长度为 ai mod ai−1a_i\bmod a_{i-1}ai​modai−1​ 的余数前缀。

余数前缀继续用同样的方式递归分解：二分找到不超过它的最大长度 aja_jaj​，把完整的 aja_jaj​ 计入 cntjcnt_jcntj​，再递归处理新的余数。
如果余数已经小于 a0=Na_0=Na0​=N，就直接说明它对应原序列的前若干个数，用差分数组给这段答案加上贡献。

每次取模后规模至少减半，所以递归次数是对数级的。
复杂度 O(N+Qlog⁡Qlog⁡max⁡qi)O(N+Q\log Q\log \max q_i)O(N+QlogQlogmaxqi​)。

AGC003F Fraction of Fractal

题意：给定一个 H×WH\times WH×W 的黑白网格，所有黑格四连通且至少有一个黑格。等级 000 的分形是一个黑格；等级 k+1k+1k+1 的分形是把等级 kkk 的分形放到原网格中每个黑格对应的位置，白格位置保持为空。给定 KKK，求等级 KKK 的分形中黑格连通块数量，对 109+710^9+7109+7 取模。

数据范围：1≤H,W≤10001 \le H,W \le 10001≤H,W≤1000，0≤K≤10180 \le K \le 10^{18}0≤K≤1018，所有黑格四连通，且至少存在一个黑格。

题解：

Trick or Hint：

记黑格总数为 SSS。

如果 K=0,1K=0,1K=0,1，或者 左右 / 上下 都联通，那么整个图始终连通，答案就是 111。
如果 左右 / 上下 都为不联通，那么每一层只是把每个黑格替换成互不相连的若干份，答案就是 SK−1S^{K-1}SK−1。

剩下就是只有一个方向能跨边界连通，不妨看成左右能连通。
设 xxx 为模板中这个左右方向上相邻黑格对的数量，yyy 为跨左右边界的连接条数。
这里先只看“左右方向上联通”的连通块数，这个的联通块数明显是下一层总体的联通块数。
我们下面求只看左右联通的联通块数，比如说样例 222 的第一层有 444 个连通块，不看上下联通。
从一层推到下一层时，先把上一层整体复制到 SSS 个黑格里，所以答案先乘 SSS。
然后新增的跨边界合并数要减掉：第一次扩展减 xxx，第二次扩展减 xyxyxy，第三次扩展减 xy2xy^2xy2，依此类推。

如果记 gig_igi​ 为扩展了 iii 次后的答案，那么

gi=S⋅gi−1−xyi−1g_i=S\cdot g_{i-1}-xy^{i-1} gi​=S⋅gi−1​−xyi−1

最终需要的是 gK−1g_{K-1}gK−1​，展开后就是

gK−1=SK−1−x(SK−2+ySK−3+y2SK−4+⋯+yK−2)g_{K-1}=S^{K-1}-x\left(S^{K-2}+yS^{K-3}+y^2S^{K-4}+\cdots+y^{K-2}\right) gK−1​=SK−1−x(SK−2+ySK−3+y2SK−4+⋯+yK−2)

代码里用 2×22\times22×2 矩阵快速幂维护这串和。

复杂度 O(HW+log⁡K)O(HW+\log K)O(HW+logK)。
