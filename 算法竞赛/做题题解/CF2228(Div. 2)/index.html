CF2228E1 Amanojaku and Sequence (Easy Version)

题意：给定一个长度为 nnn 的数组 aaa，其中每个位置要么是非负整数，要么是 −1-1−1。一次询问给出 l,r,ml,r,ml,r,m，考虑子数组 b=[al,al+1,…,ar]b=[a_l,a_{l+1},\ldots,a_r]b=[al​,al+1​,…,ar​]。需要把其中所有 −1-1−1 替换成非负整数，非 −1-1−1 的位置保持原值，并且替换后得到的序列 ccc 满足 ∑ci=m\sum c_i=m∑ci​=m。对每个合法的 ccc，定义

f(c)=∑i=1∣c∣(∑j=1icj)2f(c)=\sum_{i=1}^{|c|}\left(\sum_{j=1}^{i}c_j\right)^2 f(c)=i=1∑∣c∣​(j=1∑i​cj​)2

也就是所有前缀和平方的和。求所有合法 ccc 的 f(c)f(c)f(c) 之和，答案对 998244353998244353998244353 取模；如果不存在合法序列，则答案为 000。

数据范围：1≤t≤1041 \le t \le 10^41≤t≤104，1≤n≤3×1051 \le n \le 3\times 10^51≤n≤3×105，本版本 q=1q=1q=1 且询问操作固定为 2 l r m，−1≤ai≤106-1 \le a_i \le 10^6−1≤ai​≤106，0≤m≤1060 \le m \le 10^60≤m≤106，所有测试用例的 nnn 之和不超过 3×1053\times 10^53×105。

题解：

对于区间中的某个位置 iii，它的前缀和可以拆成两部分：

前缀和=Ai+Xi\text{前缀和}=A_i+X_i 前缀和=Ai​+Xi​

其中：

  • AiA_iAi​：这个前缀中固定数字的和；
  • XiX_iXi​：这个前缀中变量的和。

假设在当前位置之前，包括当前位置，已经出现了 tit_iti​ 个 −1-1−1，那么：

Xi=x1+x2+⋯+xtiX_i=x_1+x_2+\cdots+x_{t_i} Xi​=x1​+x2​+⋯+xti​​

所以当前位置对答案的贡献是：

(Ai+Xi)2(A_i+X_i)^2 (Ai​+Xi​)2

展开：

(Ai+Xi)2=Ai2+2AiXi+Xi2(A_i+X_i)^2=A_i^2+2A_iX_i+X_i^2 (Ai​+Xi​)2=Ai2​+2Ai​Xi​+Xi2​

于是我们需要知道三件事：

∑1,∑Xi,∑Xi2\sum 1,\qquad \sum X_i,\qquad \sum X_i^2 ∑1,∑Xi​,∑Xi2​

在继续计算之前，先记两种常用的范德蒙德卷积形式。

另外还会用到组合数的吸收恒等式：

x(x+r−1r−1)=r(x+r−1r)x\binom{x+r-1}{r-1}=r\binom{x+r-1}{r} x(r−1x+r−1​)=r(rx+r−1​)

1. 上标固定型

形式是：

∑i(ai)(bk−i)=(a+bk)\sum_i \binom{a}{i}\binom{b}{k-i}=\binom{a+b}{k} i∑​(ia​)(k−ib​)=(ka+b​)

它的特点是组合数上标固定，下标随枚举变量变化：

(ai)(bk−i)\binom{a}{i}\binom{b}{k-i} (ia​)(k−ib​)

组合意义证明：把 a+ba+ba+b 个物品分成两组，第一组有 aaa 个，第二组有 bbb 个。现在要从总共 a+ba+ba+b 个物品中选出 kkk 个，右边直接数方案数，为 (a+bk)\binom{a+b}{k}(ka+b​)。左边则枚举从第一组中选 iii 个，那么第二组就必须选 k−ik-ik−i 个，对所有 iii 求和，得到同一个方案数。

2. 下标固定型

形式是：

∑i(ia)(n−ib)=(n+1a+b+1)\sum_i \binom{i}{a}\binom{n-i}{b}=\binom{n+1}{a+b+1} i∑​(ai​)(bn−i​)=(a+b+1n+1​)

它的特点是组合数下标固定，上标随枚举变量变化：

(ia)(n−ib)\binom{i}{a}\binom{n-i}{b} (ai​)(bn−i​)

组合意义证明：右边是在 0,1,…,n0,1,\ldots,n0,1,…,n 这 n+1n+1n+1 个位置中选出 a+b+1a+b+1a+b+1 个数。考虑选出的数中第 a+1a+1a+1 小的数，设它为 iii。那么在 iii 左边的 0,1,…,i−10,1,\ldots,i-10,1,…,i−1 中需要选 aaa 个数，在 iii 右边的 i+1,…,ni+1,\ldots,ni+1,…,n 中需要选 bbb 个数，贡献为 (ia)(n−ib)\binom{i}{a}\binom{n-i}{b}(ai​)(bn−i​)。枚举这个分界数 iii，就得到左边求和。

回到本题：先数合法填法数

假设当前询问区间里一共有 KKK 个未确定位置，并且这些位置最后填入的数总和为 NNN，那么所有可行填法数量为：

∑1=(N+K−1K−1)\sum 1=\binom{N+K-1}{K-1} ∑1=(K−1N+K−1​)

这是把 NNN 分给 KKK 个非负整数变量的方案数。

固定一个位置，拆左右两边

现在固定区间中的某个位置 iii。设它之前，包括它自己，已经出现了 k1k_1k1​ 个未确定位置，这些位置的变量和为 n1n_1n1​；它之后有 k2k_2k2​ 个未确定位置，变量和为 n2n_2n2​。于是：

k1+k2=K,n1+n2=Nk_1+k_2=K,\qquad n_1+n_2=N k1​+k2​=K,n1​+n2​=N

并且 Xi=n1X_i=n_1Xi​=n1​。

计算 ∑Xi\sum X_i∑Xi​

先计算 ∑Xi\sum X_i∑Xi​。枚举 n1n_1n1​，则前半部分的填法数为 (n1+k1−1k1−1)\binom{n_1+k_1-1}{k_1-1}(k1​−1n1​+k1​−1​)，后半部分的填法数为 (n2+k2−1k2−1)\binom{n_2+k_2-1}{k_2-1}(k2​−1n2​+k2​−1​)，所以：

∑Xi=∑n1n1(n1+k1−1k1−1)(n2+k2−1k2−1)=∑n1k1(n1+k1−1k1)(n2+k2−1k2−1)=k1(N+K−1K)\begin{aligned} \sum X_i &=\sum_{n_1} n_1 \binom{n_1+k_1-1}{k_1-1} \binom{n_2+k_2-1}{k_2-1} \\ &=\sum_{n_1} k_1 \binom{n_1+k_1-1}{k_1} \binom{n_2+k_2-1}{k_2-1} \\ &=k_1\binom{N+K-1}{K} \end{aligned} ∑Xi​​=n1​∑​n1​(k1​−1n1​+k1​−1​)(k2​−1n2​+k2​−1​)=n1​∑​k1​(k1​n1​+k1​−1​)(k2​−1n2​+k2​−1​)=k1​(KN+K−1​)​

最后一步用的是下标固定型范德蒙德卷积。

计算 ∑Xi2\sum X_i^2∑Xi2​

再计算 ∑Xi2\sum X_i^2∑Xi2​。同样枚举 n1n_1n1​：

∑Xi2=∑n1n12(n1+k1−1k1−1)(n2+k2−1k2−1)\sum X_i^2 =\sum_{n_1} n_1^2 \binom{n_1+k_1-1}{k_1-1} \binom{n_2+k_2-1}{k_2-1} ∑Xi2​=n1​∑​n12​(k1​−1n1​+k1​−1​)(k2​−1n2​+k2​−1​)

先像计算 ∑Xi\sum X_i∑Xi​ 时一样吸收掉一个 n1n_1n1​：

∑Xi2=∑n1n12(n1+k1−1k1−1)(n2+k2−1k2−1)=k1∑n1n1(n1+k1−1k1)(n2+k2−1k2−1)\begin{aligned} \sum X_i^2 &=\sum_{n_1} n_1^2 \binom{n_1+k_1-1}{k_1-1} \binom{n_2+k_2-1}{k_2-1} \\ &=k_1\sum_{n_1} n_1 \binom{n_1+k_1-1}{k_1} \binom{n_2+k_2-1}{k_2-1} \end{aligned} ∑Xi2​​=n1​∑​n12​(k1​−1n1​+k1​−1​)(k2​−1n2​+k2​−1​)=k1​n1​∑​n1​(k1​n1​+k1​−1​)(k2​−1n2​+k2​−1​)​

接下来把剩下的 n1n_1n1​ 凑成 (n1−1)+1(n_1-1)+1(n1​−1)+1，其中 +1+1+1 的部分正好对应一份 ∑Xi\sum X_i∑Xi​：

∑Xi2=k1∑n1(n1−1)(n1+k1−1k1)(n2+k2−1k2−1)+∑Xi=k1(k1+1)∑n1(n1+k1−1k1+1)(n2+k2−1k2−1)+∑Xi=k1(k1+1)(N+K−1K+1)+k1(N+K−1K)\begin{aligned} \sum X_i^2 &=k_1 \sum_{n_1}(n_1-1) \binom{n_1+k_1-1}{k_1} \binom{n_2+k_2-1}{k_2-1} +\sum X_i \\ &=k_1(k_1+1) \sum_{n_1} \binom{n_1+k_1-1}{k_1+1} \binom{n_2+k_2-1}{k_2-1} +\sum X_i \\ &=k_1(k_1+1)\binom{N+K-1}{K+1} +k_1\binom{N+K-1}{K} \end{aligned} ∑Xi2​​=k1​n1​∑​(n1​−1)(k1​n1​+k1​−1​)(k2​−1n2​+k2​−1​)+∑Xi​=k1​(k1​+1)n1​∑​(k1​+1n1​+k1​−1​)(k2​−1n2​+k2​−1​)+∑Xi​=k1​(k1​+1)(K+1N+K−1​)+k1​(KN+K−1​)​

合并当前位置的贡献

记

C0=(N+K−1K−1),C1=(N+K−1K),C2=(N+K−1K+1).C_0=\binom{N+K-1}{K-1},\qquad C_1=\binom{N+K-1}{K},\qquad C_2=\binom{N+K-1}{K+1}. C0​=(K−1N+K−1​),C1​=(KN+K−1​),C2​=(K+1N+K−1​).

固定位置 iii 时，AiA_iAi​ 和 k1k_1k1​ 都已经确定，枚举所有合法填法后：

∑(Ai+Xi)2=Ai2∑1+2Ai∑Xi+∑Xi2=Ai2C0+2Aik1C1+k1(k1+1)C2+k1C1.\begin{aligned} \sum (A_i+X_i)^2 &=A_i^2\sum 1+2A_i\sum X_i+\sum X_i^2\\ &=A_i^2C_0+2A_i k_1C_1+k_1(k_1+1)C_2+k_1C_1. \end{aligned} ∑(Ai​+Xi​)2​=Ai2​∑1+2Ai​∑Xi​+∑Xi2​=Ai2​C0​+2Ai​k1​C1​+k1​(k1​+1)C2​+k1​C1​.​

所以从左到右扫一遍询问区间即可。维护两个量：

  • sumA：当前前缀中固定数字的和，也就是 AiA_iAi​；
  • k1：当前前缀中未确定位置的个数。

如果当前位置是固定数字，就把它加入 sumA；如果当前位置是 −1-1−1，就让 k1 加一。然后把上面的式子加进答案。

特别地，若 N<0N<0N<0，说明固定数字的和已经超过 mmm，没有合法填法，所有组合数函数都会返回 000。若 K=0K=0K=0，则不存在变量：只有当 N=0N=0N=0 时有唯一一种填法，此时需要手动令 C0=1C_0=1C0​=1，其余和变量有关的项都会因为 k1=0k_1=0k1​=0 而消失。

CF2228E2 Amanojaku and Sequence (Hard Version)

题意：给定长度为 nnn 的数组 aaa，其中每个位置要么是非负整数，要么是 −1-1−1。定义

f(c)=∑i=1∣c∣(∑j=1icj)2f(c)=\sum_{i=1}^{|c|}\left(\sum_{j=1}^{i}c_j\right)^2 f(c)=i=1∑∣c∣​(j=1∑i​cj​)2

也就是所有前缀和平方的和。现在有 qqq 次操作：

  • 1 p v：将 apa_pap​ 修改为 vvv；
  • 2 l r m：考虑子数组 [al,…,ar][a_l,\ldots,a_r][al​,…,ar​]，把其中所有 −1-1−1 替换成非负整数，非 −1-1−1 的位置保持原值，并且替换后序列 ccc 的元素和为 mmm。求所有合法 ccc 的 f(c)f(c)f(c) 之和，答案对 998244353998244353998244353 取模；如果不存在合法序列，则答案为 000。

数据范围：1≤t≤1041 \le t \le 10^41≤t≤104，1≤n,q≤3×1051 \le n,q \le 3\times 10^51≤n,q≤3×105，−1≤ai,v≤106-1 \le a_i,v \le 10^6−1≤ai​,v≤106，0≤m≤1060 \le m \le 10^60≤m≤106，1≤p≤n1 \le p \le n1≤p≤n，1≤l≤r≤n1 \le l \le r \le n1≤l≤r≤n，所有测试用例的 nnn 之和不超过 3×1053\times 10^53×105，qqq 之和不超过 3×1053\times 10^53×105。

题解：

沿用 E1 的推式。对于一次询问，设区间中固定数的前缀和为 AiA_iAi​，−1-1−1 的前缀个数为 KiK_iKi​，最后答案只和下面几类区间统计量有关：

∑Ai,∑Ki,∑Ai2,∑Ki2,∑AiKi.\sum A_i,\quad \sum K_i,\quad \sum A_i^2,\quad \sum K_i^2,\quad \sum A_iK_i. ∑Ai​,∑Ki​,∑Ai2​,∑Ki2​,∑Ai​Ki​.

因此把每个位置看成二元组 (A,K)(A,K)(A,K)：若 ai=−1a_i=-1ai​=−1，则 (A,K)=(0,1)(A,K)=(0,1)(A,K)=(0,1)；否则 (A,K)=(ai,0)(A,K)=(a_i,0)(A,K)=(ai​,0)。用线段树维护上面信息即可

CF2228F Momoyo and the Network

题意：给定一棵 nnn 个点的树，每个点有权值 aia_iai​。选择一条恰好包含 kkk 条边的简单路径，并删除路径上的所有边，树会被分成 k+1k+1k+1 个连通块。每个连通块的权值为其中所有点权之和，要求最大化这些连通块权值的最小值；如果不存在长度为 kkk 的路径，则输出 −1-1−1。

数据范围：1≤t≤1041 \le t \le 10^41≤t≤104，2≤n≤2×1052 \le n \le 2\times 10^52≤n≤2×105，1≤k≤n−11 \le k \le n-11≤k≤n−1，1≤ai≤1091 \le a_i \le 10^91≤ai​≤109，所有测试用例的 nnn 之和不超过 2×1052\times 10^52×105。

题解：

最开始想的是二分答案 xxx 加点分治。

对于当前点分治的根 uuu，考虑它的每个儿子方向。先找出在所有切出来的连通块权值都不小于 xxx 的限制下，这个方向最多能延伸出的链长。

这里还需要知道从 uuu 走向某个相邻点 vvv 后，对应那一侧连通块的权值。可以先随便定根预处理 sub，然后：

  • 如果 vvv 是 uuu 的儿子，这一侧权值是 sub[v]。
  • 否则这一侧权值是 sum - sub[u]。

接着尝试把两个不同儿子方向的最长链拼起来。如果拼出的长度至少为 kkk，并且路径删掉后所有连通块的权值都满足条件，那么当前二分值可行。

实现上可以把每个儿子方向得到的 (len, sum) 排序，然后用后缀最小值维护可拼接的另一条链。每个点只会作为某一层点分治根的儿子被排序一次，所以排序部分不会额外炸复杂度。

不过这个做法细节很多，也比较容易写挂。

而且也过不去，会TLE，不如直接点分治好写。

其实可以直接点分治，不需要二分答案。

对于当前点分治根 uuu，考虑答案路径经过 uuu 的情况。路径会从 uuu 的一个相邻方向伸出去，或者从两个不同相邻方向拼起来。点分治保证每条路径都会在某一层被它经过的分治根统计到。

先预处理每条有向边对应那一侧连通块的权值。随便以 111 为根求 subsubsub，若 vvv 是 uuu 的儿子，则从 uuu 走向 vvv 的那一侧权值为 subvsub_vsubv​；否则为 sum−subusum-sub_usum−subu​。代码里把这些信息存在 vecuvec_uvecu​ 里。

然后从某个方向出发 DFS 收集链。设当前链长为 lenlenlen，当前这一步走进来的连通块权值为 nownownow，mnmnmn 表示沿途已经切出来的连通块权值最小值，那么 usedlenused_{len}usedlen​ 记录这种长度下 min⁡(now,mn)\min(now,mn)min(now,mn) 的最大值。也就是：只看这一个方向内部，长度为 lenlenlen 的半条链最多能保证多大的最小连通块权值。

合并时枚举当前方向的一条半链，另一侧需要的长度就是 k−lenk-lenk−len。如果之前已经有这个长度的最优半链，就可以用两边的最小值更新答案。

真正麻烦的是，删掉两个方向后，包含根 uuu 的剩余部分也会成为一个连通块，它也要参与取最小值。这里把根的相邻方向按连通块权值从大到小排序，单独处理最大的方向 bigbigbig。

如果路径删掉的两个方向都不是最大方向，那么剩余部分一定还包含最大方向，权值相对更大，可以先不把它作为瓶颈，直接合并普通方向。

接着处理经过最大方向的路径。先把其他方向合成 optoptopt，其中 optlenopt_{len}optlen​ 表示从非最大方向拿长度为 lenlenlen 时，连同“切掉最大方向和当前方向后的剩余部分”一起考虑后的最优值。最后再枚举最大方向的链长，和 optk−lenopt_{k-len}optk−len​ 合并。

这个做法可以省掉二分，但实现细节还是比较多。
