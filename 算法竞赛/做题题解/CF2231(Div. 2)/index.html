CF2231E Graph Cutting

题意：给定一棵 nnn 个点的树和一个整数 ddd。选择三个互不相同的点 a,b,ca,b,ca,b,c，取树上包含这三个点的最小连通子图，也就是三条两两路径的并集。问有多少个三元组 (a,b,c)(a,b,c)(a,b,c)，使得这个连通子图恰好包含 ddd 个点。

数据范围：1≤t≤5001 \le t \le 5001≤t≤500，3≤d≤n≤20003 \le d \le n \le 20003≤d≤n≤2000，所有测试用例的 nnn 之和不超过 200020002000。

题解：

做法一：

枚举三元组中的一个点作为根 rtrtrt，再选择另外两个点 x,yx,yx,y。若深度从 111 开始计算，则 rt,x,yrt,x,yrt,x,y 生成的连通块大小为

depx+depy−deplca⁡(x,y).dep_x+dep_y-dep_{\operatorname{lca}(x,y)}. depx​+depy​−deplca(x,y)​.

DFS 时枚举 lca⁡(x,y)=u\operatorname{lca}(x,y)=ulca(x,y)=u，用 dsu on tree 维护已经处理过的儿子子树中各个深度的点数。处理 uuu 的儿子 vvv 时，遍历 vvv 子树中的点 xxx，只需统计深度为

depy=d+depu−depxdep_y=d+dep_u-dep_x depy​=d+depu​−depx​

的点 yyy，这些点必须来自之前处理过的其他儿子子树。

还要单独统计 x,yx,yx,y 中有一个等于 uuu 的情况。最后每个三元组会因为三个点分别作为根而被统计 333 次，所以答案除以 333。复杂度为 O(n2log⁡n)O(n^2\log n)O(n2logn)。

做法二：树上背包

也可以直接对连通块做树上背包。任意定根，设 fu,d,xf_{u,d,x}fu,d,x​ 表示在 uuu 子树内选出一个大小为 ddd、包含 uuu 的连通块，并且其中恰好有 xxx 个点被选为三元组中的点。初始化为 fu,1,0=fu,1,1=1f_{u,1,0}=f_{u,1,1}=1fu,1,0​=fu,1,1​=1。

合并儿子 vvv 时做普通背包转移：

nfdu+dv,x+y+=fu,du,x⋅fv,dv,y.nf_{du+dv,x+y} \mathrel{+}= f_{u,du,x}\cdot f_{v,dv,y}. nfdu+dv,x+y​+=fu,du,x​⋅fv,dv,y​.

接上来的部分必须包含 vvv，才能通过边 (u,v)(u,v)(u,v) 连起来，所以代码里合并儿子时要求 y≥1y\ge 1y≥1。同时不需要合并 y=3y=3y=3 的状态，因为三个点已经全部在 vvv 子树内，再接上 uuu 只会得到更大的非最小连通块。每个点 uuu 处理完后，将 fu,d,3f_{u,d,3}fu,d,3​ 加入答案即可。

CF2231F Quadratic Jumps

题意：有 nnn 个点 1∼n1\sim n1∼n，若 ∣i−j∣|i-j|∣i−j∣ 是完全平方数，则 i,ji,ji,j 之间有一条边。给定 qqq 次询问 (a,b)(a,b)(a,b)，求 aaa 到 bbb 的最短路长度。

数据范围：1≤t≤10001\le t\le 10001≤t≤1000，2≤n≤2⋅1052\le n\le 2\cdot 10^52≤n≤2⋅105，1≤q≤1051\le q\le 10^51≤q≤105，1≤a<b≤n1\le a<b\le n1≤a<b≤n。所有测试用例的 nnn 之和不超过 2⋅1052\cdot 10^52⋅105，qqq 之和不超过 10510^5105。

题解：

两点之间每次跳的距离必须是完全平方数。由四平方和定理，任意正整数都能表示成不超过四个平方数之和，所以答案最多为 444。因此只需要判断答案是否为 1,2,31,2,31,2,3，都不满足则为 444。

记 d=b−ad=b-ad=b−a。

答案为 111：只需判断 ddd 是否为完全平方数，可以 O(1)O(1)O(1) 完成。

答案为 222：

  • 同向：d=p2+q2d=p^2+q^2d=p2+q2。平方数只有 O(n)O(\sqrt n)O(n​) 个，预处理所有两平方数之和即可。
  • 反向：形如 d=p2−q2d=p^2-q^2d=p2−q2。也就是先往一侧跳远一点，再往回跳短一点。

反向情况需要检查中间点是否越界。若向右再回退，则需要 a+p2≤na+p^2\le na+p2≤n；若向左再跳回，则需要 b−p2≥1b-p^2\ge 1b−p2≥1。所以对询问 (a,b)(a,b)(a,b)，只要满足

p2≤max⁡(n−a,b−1)p^2\le \max(n-a,b-1) p2≤max(n−a,b−1)

即可选择其中一个方向完成两步跳。

于是预处理 vis2[k][d]，表示是否存在 p≤kp\le kp≤k 和 q<pq<pq<p，使得 d=p2−q2d=p^2-q^2d=p2−q2。预处理时对 kkk 做前缀或，这样查询反向两步时令

k=⌊max⁡(n−a,b−1)⌋k=\left\lfloor\sqrt{\max(n-a,b-1)}\right\rfloor k=⌊max(n−a,b−1)​⌋

直接判断 vis2[k][d] 即可。

答案为 333：枚举第一步从 aaa 跳到的位置。第一步可以向左或向右跳 i2i^2i2，共有 O(n)O(\sqrt n)O(n​) 种合法位置。设跳到 xxx，如果 check2(x,b) 成立，则原询问答案为 333。

预处理后，每个询问枚举第一步，复杂度 O(n)O(\sqrt n)O(n​)，总复杂度 O(qn)O(q\sqrt n)O(qn​)。
