CF2229D Me When Median Problem

CF2229E Deconstruction Tree

题意：给定一棵 nnn 个点的树和一个初始为空的集合 SSS。重复 n−1n-1n−1 次：先找到当前树中编号最大的叶子 xxx，把 xxx 加入 SSS，然后选择一个不同于 xxx 的叶子并删除。问最终能够得到多少种不同的集合 SSS，答案对 998244353998244353998244353 取模。

数据范围：1≤t≤1041 \le t \le 10^41≤t≤104，2≤n≤2×1052 \le n \le 2\times 10^52≤n≤2×105，所有测试用例的 nnn 之和不超过 2×1052\times 10^52×105。

题解：

以最大的编号点 nnn 为根。

直接考虑一个点能转移到哪里不太好做，反过来考虑一个点可以从哪些状态转移过来。记 mxumx_umxu​ 为 uuu 子树内除 uuu 外的最大编号点，如果当前最大叶子是 iii，那么想让 iii 被加入集合，必须先删掉它子树里所有比它编号大的点，所以需要满足 i>mxii>mx_ii>mxi​。

设 fif_ifi​ 表示当前最大叶子第一次变成 iii 的方案数，那么它只能从 mxi+1mx_i+1mxi​+1 到 i−1i-1i−1 这些编号转移过来：

fi=∑j=mxi+1i−1fj.f_i=\sum_{j=mx_i+1}^{i-1} f_j. fi​=j=mxi​+1∑i−1​fj​.

用前缀和维护：

fi=prefi−1−prefmxi.f_i=pref_{i-1}-pref_{mx_i}. fi​=prefi−1​−prefmxi​​.

然后考虑根节点什么时候可以最后变成叶子。只要保留根的一棵子树，把其他子树全部删光，根就会成为叶子。所以如果 iii 所在根子树之外的最大编号小于 iii，那么 fif_ifi​ 就可以计入答案。

如果 nnn 一开始就是叶子，答案显然为 111。

CF2229F Load Unbalancing

题意：给定长度为 nnn 的数组 aaa 和整数 kkk。可以先任意重排 aaa，之后从前往后依次把 aia_iai​ 加到当前长度为 kkk 的数组 bbb 中的某个最小值位置，初始 bbb 全为 000。定义 f(a)=max⁡bif(a)=\max b_if(a)=maxbi​，求所有重排中 f(a)f(a)f(a) 的最大值。

数据范围：1≤t≤1041 \le t \le 10^41≤t≤104，1≤k≤n≤181 \le k \le n \le 181≤k≤n≤18，1≤ai≤1091 \le a_i \le 10^91≤ai​≤109，所有测试用例的 2n2^n2n 之和不超过 2182^{18}218。

题解：

首先贪心地想，最大值可以最后加入，这样一定不劣。于是先把最大值拿出来，最后答案再加上它。

然后二分最后一次加入最大值之前，所有桶的最小值能否都达到 xxx。因为最大值最后一定会被加到当前最小的位置上，所以如果在此之前已经能让 kkk 个桶都至少为 xxx，那么最终最大值就是 x+max⁡aix+\max a_ix+maxai​。

这样问题就变成：把剩下的数按某种顺序依次放入当前桶，当前桶达到 xxx 后就换下一个桶，问能否填满至少 kkk 个桶。这个过程和原题中“每次加到最小值位置”的过程是等价的。

然后做状压 DP。令 dp[sta] 表示已经使用集合 sta 中的数时，最优状态为：

  • first：已经填满的桶数；
  • second：当前桶里的和。

枚举最后加入的是哪个数，从 pre = sta ^ (1 << bit) 转移过来。pair 的字典序比较正好就是先最大化填满桶数，再最大化当前桶里的和。

总复杂度 O(n2nlog⁡V)O(n2^n\log V)O(n2nlogV)。
