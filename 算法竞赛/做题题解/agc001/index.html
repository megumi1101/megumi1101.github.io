AGC001C Shorten Diameter

题意：给定一棵有 NNN 个点的树，点间距离定义为简单路径上的边数。你可以删除一些点，但每次删除后剩余部分都必须保持连通（等价于只能删除叶子）。目标是让剩余树的直径不超过 KKK，求最少需要删除多少个点。

数据范围：2≤N≤20002 \le N \le 20002≤N≤2000，1≤K≤N−11 \le K \le N-11≤K≤N−1，输入保证是一棵树。

题解：

Trick or Hint：枚举最后最终树的中心。

思路是枚举最终子树的“中心”：当 KKK 为偶数时枚举中心点，当 KKK 为奇数时枚举中心边。对于每个中心，统计在允许半径内最多能保留多少个点。
所有方案中保留点数取最大值，答案就是 N−max⁡(保留点数)N-\max(\text{保留点数})N−max(保留点数)。
复杂度 O(N2)O(N^2)O(N2)。

AGC001D Arrays and Palindrome

题意：给定长度为 MMM 的序列 AAA，且 ∑Ai=N\sum A_i=N∑Ai​=N。需要构造两个正整数序列 a,ba,ba,b，其中 aaa 是 AAA 的一个重排，并且满足 ∑ai=∑bi=N\sum a_i=\sum b_i=N∑ai​=∑bi​=N。要求任意长度为 NNN 的序列，如果它同时满足“按 aaa 分块后每块都是回文”和“按 bbb 分块后每块都是回文”，那么这个序列的所有元素都必须相等。输出一组可行的 a,ba,ba,b，无解则输出 Impossible。

数据范围：1≤N≤1051 \le N \le 10^51≤N≤105，1≤M≤1001 \le M \le 1001≤M≤100，1≤Ai≤1051 \le A_i \le 10^51≤Ai​≤105，且 ∑Ai=N\sum A_i=N∑Ai​=N。

题解：

Trick or Hint：把回文块约束看成连边。

先转化一下题意。可以把每个回文块看成若干条连边，也就是把块内被要求相等的位置连起来。
于是问题等价于：按 aaa 和 bbb 分块连边后，这 NNN 个位置必须整体连通。
要让 NNN 个点连通，至少需要 N−1N-1N−1 条边；换到分块的视角，就是 aaa 和 bbb 中总共最多只能有两个奇数长度的块。
构造时可以让两种分块整体错开一位，并把 aaa 中的奇数长度块放到两端。

AGC001E BBQ Hard

题意：给定 NNN 组 (Ai,Bi)(A_i,B_i)(Ai​,Bi​)，要求计算

∑i=1N∑j=i+1N(Ai+Aj+Bi+BjAi+Aj)\sum_{i=1}^{N}\sum_{j=i+1}^{N}\binom{A_i+A_j+B_i+B_j}{A_i+A_j} i=1∑N​j=i+1∑N​(Ai​+Aj​Ai​+Aj​+Bi​+Bj​​)

结果对 109+710^9+7109+7 取模。

数据范围：2≤N≤2×1052 \le N \le 2\times 10^52≤N≤2×105，1≤Ai,Bi≤20001 \le A_i,B_i \le 20001≤Ai​,Bi​≤2000。

题解：

Solution1:

Trick or Hint：把组合数看成格路计数，用二维 DP 汇总所有有序点对后减去自配对再除以二。

可以把 (Ai+Aj+Bi+BjAi+Aj)\binom{A_i+A_j+B_i+B_j}{A_i+A_j}(Ai​+Aj​Ai​+Aj​+Bi​+Bj​​) 看成从 (−Ai,−Bi)(-A_i, -B_i)(−Ai​,−Bi​) 走到 (Aj,Bj)(A_j, B_j)(Aj​,Bj​) 的路径数，每一步只能向右或向上走。
于是把所有 (−Ai,−Bi)(-A_i,-B_i)(−Ai​,−Bi​) 的初值都设为 111，做一遍二维 DP，就能求出所有有序点对 (i,j)(i,j)(i,j) 的贡献和。
最后把 i=ji=ji=j 的贡献减掉，再除以 222，就是题目要求的无序点对答案。

Solution2:

Trick or Hint：把组合数用OGF改写成取常数项。

接下来只需要求出

P(x)=∑k=1mQk(1+x)kP(x)=\sum_{k=1}^{m}Q_k(1+x)^k P(x)=k=1∑m​Qk​(1+x)k

并取 P(x)2P(x)^2P(x)2 的常数项。

因为 QkQ_kQk​ 里有负次数项，不方便直接用数组存，所以可以整体平移一下，令 Q^k=QkxV\hat{Q}_k=Q_kx^VQ^​k​=Qk​xV。
这样求出的多项式也会整体乘上 xVx^VxV，最后在平方时取对应位置的系数即可。
